Dog or Cat?

Neural Network Dog/Cat Image Classifier

Instructions

Upload an image containing a dog or a cat and the neural network will tell you which it contains.

About this Tool

The neural network underpinning this website was developed in Python by James Bridgwater Court based upon a walkthrough by YouTube user Siddhardhan, using the dataset for the Kaggle Dogs vs Cats competition and employing transfer learning with the MobileNetV2 image classification model from Keras. The front-end is implemented using Flask.
